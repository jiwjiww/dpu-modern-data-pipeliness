# โปรเจค AQI Data Pipeline

## วัตถุประสงค์
- วิเคราะห์คุณภาพอากาศ
- สร้าง Data Pipeline อัตโนมัติ

## เครื่องมือที่ใช้
- Python
- Apache Airflow
- Pandas
- Matplotlib

## วิธีติดตั้ง
1. Clone Repository
2. สร้าง Virtual Environment
3. pip install -r requirements.txt
4. รัน Airflow

## ผลลัพธ์
- รายงานคุณภาพอากาศ
- กราฟแนวโน้ม AQI
